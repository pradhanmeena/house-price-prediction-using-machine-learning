## Project Title
### House Price Prediction Using Machine Learning 
## Description
`House Price Prediction` is a Python-based software application that **uses machine learning** techniques to predict house prices based on various factors `(such as location, plot size, house size, number of bedrooms, number of bathrooms, house condition, build year, and basement availability)`. The main purpose of this project is to accurately estimate the price of a house based on these factors, which can be helpful for both buyers and sellers. Internally, the model analyzes historical data and predicts the price based on the given input data.

## Usage
Follow the steps below to use this software:
1. Open the project folder and run the file `House_Price_prediction.py`.
2. In the GUI of software click on browse and open `dataset` folder.
3. Select Dataset `House_Price_prediction.csv` and click **Submit** button.
4. Now you need to fill details correctly and just click on predict.
